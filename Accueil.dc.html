À RETENIR

Dans une chaîne commerciale, la marchandise et l'argent ne voyagent jamais dans le même sens. Le produit descend vers le client ; le paiement remonte vers le fournisseur. Les documents, eux, circulent dans les deux sens pour garder une trace écrite de chaque échange.
